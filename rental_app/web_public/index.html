如何使用

  1. 在上方「主功能」选好入口：左为房源分析，右为合同分析（可先读本页引导）
  2. 可选：打开「智能入口」用自然语言写需求并保存草稿，再进入房源或合同流程
  3. 房源分析：输入一句租房需求并提交
  4. 查看推荐结果和风险提示
  5. 收藏想要的房源
  6. 从「分析历史」或对比页继续决策（房源列表在分析历史 → 房源分区）
  7. 审合同：从主功能右卡、顶栏或表单下「去合同分析」进入，上传或粘贴租约
