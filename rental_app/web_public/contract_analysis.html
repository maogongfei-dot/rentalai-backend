上传合同（推荐）

支持 .txt、.pdf、.docx；单文件默认上限 15 MB（前后端一致，可由环境变量 RENTALAI_CONTRACT_UPLOAD_MAX_BYTES 调整）。点击「提交分析」将以 multipart 上传至服务器并分析。

开发：服务端路径（非上传）

不经过浏览器上传，由 API 直接读服务器磁盘路径（须与 API 进程同机可读，如相对 rental_app 的 sample）。生产环境需额外鉴权与路径白名单，此处仅供本地联调。
